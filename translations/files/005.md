# 寂寞夜晚

|         |         |
|---------|---------|
| 寂寞夜晚　如常平淡 | Lonely night, calm as usual |
| 望著天花板　告別那份浪漫 | Staring at the ceiling, good bye to my love |
| 續漸習慣　空空臂彎 | Getting used to the empty arms |
| 過去總不返　告別那份浪漫 | Let bygones be bygones, and good bye to my love |
| 誰話過海枯石爛　誰共我相擁每晚 | Who ever said forever, who held me through the nights |
| 誰讓熱愛瀰漫　但現在影孤隻單 | Who let the love diffuse, but then left me alone |
| 當天多麼激情　今天卻是冷 | How intense it was, and how cold it is now |
| 冷雨濕我身　身軀已習慣 | Let the cold rain be, already got used to it |
| 慣了孤寂　寂寞又繼續瀰漫 | Solitude as always, and the feeling ferments again. |
